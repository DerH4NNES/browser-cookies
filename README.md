# tiny-cookies
CommonJS cookies library for the browser - **under development**

Features:
  - ~0.5kb minified and gzipped
  - RFC6265 compliant
  - Extensive unit test coverage

### Browser compatability
Goal:
  - Internet Explorer 6+
  - Chrome 1+
  - Firefox 3+
  - Safari 4+
  - Opera 10+
  - Android
  - iOS
